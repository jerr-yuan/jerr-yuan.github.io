PHILOSOPHY

为什么做这个知识库

把每次学习转成可复用的资产。我每天和 Claude Code 在贷后风控、AI 虚拟坐席的项目里推进真实工程，过程中沉淀了大量"非教科书但比教科书更实用"的认知。这些认知如果只留在对话历史里，会随时间消失。所以我把它们整理成 HTML 知识库——既给未来的自己复习用，也作为"懂 LLM 训练原理"这件事的对外展示。

每章遵循统一结构：定义 → 类比 → 对比表 → 业务联系。我不写教科书翻译，只写自己真的搞清楚的东西。
不在范围：纯数学推导、最新论文综述。
